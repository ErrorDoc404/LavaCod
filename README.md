# LavaCod

LavaCod is advance discord music bot with button interaction

# Installation

Use the package manager [Node v16](https://nodejs.org/en/download/) to install diapedesis in project.

```bash
npm install
```

## Config

make .env file

```bash
touch .env
```

.env file

```env
TOKEN=
Discord_ClientID=
MONGOOSE_URL=

LAVALINK_HOST=
LAVALINK_PORT=
LAVALINK_PASSWORD=
LAVALINK_SECURE=
```

## Run
```bash
node .
```
